# 위치기반서비스사업 신고 준비 체크리스트

## 1. 신고 근거 및 관할

- **근거 법령**: 위치정보의 보호 및 이용 등에 관한 법률 (위치정보법) 제9조
- **관할 기관**: 방송통신위원회 (온라인 신고 가능)
- **신고 사이트**: [방통위 위치정보 통합지원 사이트](https://law.kcc.go.kr/) 또는 정부24
- **처리 기간**: 통상 신고 접수 후 30일 이내 (보완 요청 시 별도 기간)
- **수수료**: 없음 (신고 대상은 사업자 단위 무료, 등록·허가와 다름)

## 2. 사업 분류 확인

| 구분 | 대상 | PawMate 해당 여부 |
|---|---|---|
| 위치정보사업 (허가) | 직접 위치정보를 수집·제공하는 자 (예: 통신사) | ❌ 해당 없음 |
| **위치기반서비스사업 (신고)** | **위치정보를 이용하여 서비스를 제공하는 자** | **✅ 신고 대상** |
| 개인위치정보사업자 | 개인위치정보를 수집·제공하는 자 | ❌ 해당 없음 |

## 3. 사업자 요건 확인

- [ ] 사업자등록증 (개인 또는 법인) 발급
- [ ] 대표자 신원 확인 서류
- [ ] 개인정보 보호책임자 지정
- [ ] 위치정보관리책임자 지정 (개인정보 보호책임자 겸직 가능)

### 4-7. 위치정보 보호조치 자체 점검 결과

방통위·KISA에서 배포하는 「위치정보 보호조치 준수 여부 자체 점검표」에 따라 점검:

- [ ] 관리적 보호조치 (내부관리계획, 접근권한, 교육)
- [ ] 기술적 보호조치 (암호화, 접근제어, 로그 보관)
- [ ] 물리적 보호조치 (전산실 접근 통제 — 클라우드 이용 시 위탁사 확인서)
- [ ] 위탁계약서 사본 (Supabase 등 클라우드 서비스 위탁 관련)

## 5. 신고 이후 지속 준수 사항

### 5-1. 위치정보 이용·제공 사실 확인자료 보관

- [ ] 6개월 이상 자동 기록·보관하는 시스템 구축 완료 확인
- [ ] 이용자의 열람 요구 시 지체 없이 제공하는 절차 마련

### 5-2. 이용자 권리 대응 체계

- [ ] 개인위치정보 열람·정정·삭제 요구에 대한 처리 절차 문서화
- [ ] 동의 철회 시 즉시 파기하는 시스템 검증
- [ ] 위치정보관리책임자 연락처를 앱 내·홈페이지에 상시 게시

### 5-3. 사업자 정보 변경 시 변경신고

다음 사항 변경 시 30일 이내 방통위에 변경신고:

- 상호·명칭
- 대표자
- 주된 사무소 소재지
- 사업의 종류

### 5-4. 정기 점검 및 보고

- [ ] 연 1회 이상 자체 보호조치 점검 실시
- [ ] 개인정보 취급자 정기 교육 실시
- [ ] 위치정보 침해사고 발생 시 즉시 방통위·KISA 신고

## 6. 벌칙 요약 (참고용)

- 신고 없이 사업 영위: 3년 이하 징역 또는 3천만원 이하 벌금
- 이용자 동의 없이 개인위치정보 수집·이용: 5년 이하 징역 또는 5천만원 이하 벌금
- 위치정보 확인자료 미보관: 1천만원 이하 과태료

## 7. 추천 진행 순서

1. **[1주차]** 사업자등록증 발급 (아직 미완인 경우)
2. **[1주차]** 4개 문서(사업계획서, 위치기반 약관, 개인정보 처리방침, 서비스 약관) placeholder 확정
3. **[2주차]** 위치정보시스템 상세 구성도 작성 (Draw.io 등)
4. **[2주차]** 자체 보호조치 점검표 작성
5. **[2주차]** **행정사 또는 IT 전문 변호사 검토 의뢰** (강력 권장)
6. **[3주차]** 방통위 온라인 신고 제출
7. **[3~4주차]** 보완 요청 대응 및 신고 수리 확인
8. **[신고 완료 후]** 앱 내 약관·처리방침 정식 게시, 서비스 정식 개시
