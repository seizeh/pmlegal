# 법률 문서 초안 사용 시 주의사항

본 폴더의 모든 문서는 **PawMate 서비스 기반 초안**이며, 실제 신고·게시 전에
**반드시 행정사 또는 변호사의 최종 검토**를 받아야 합니다. 사업 형태(개인/법인),
서버 리전, 실제 수집 항목, 제3자 API 이용 범위 등에 따라 내용이 달라집니다.

## 문서 목록

| 파일 | 용도 | 시급도 |
|---|---|---|
| `01_사업계획서_초안.md` | 위치기반서비스사업 신고 시 첨부 | ★★★ |
| `02_위치기반서비스_이용약관.md` | 서비스 내 게시 + 신고 첨부 | ★★★ |
| `03_개인정보_처리방침.md` | 서비스 내 상시 게시 | ★★★ |
| `04_서비스_이용약관.md` | 서비스 내 게시 | ★★☆ |
| `05_신고_준비_체크리스트.md` | 방통위 신고 절차 안내 | ★★☆ |

## 반드시 채워야 하는 부분

문서 내 `[[대괄호]]` 는 모두 **미확정 placeholder** 입니다:
- `[[사업자명]]` / `[[대표자명]]`
- `[[사업장 주소]]` / `[[사업자등록번호]]`
- `[[대표번호]]` / `[[대표이메일]]`
- `[[시행일]]`
- `[[Supabase 리전]]` 예: `ap-northeast-2 (서울)`
- `[[개인정보보호책임자]]` 이름·연락처

## 규제 근거

- **위치정보법** (위치정보의 보호 및 이용 등에 관한 법률)
- **개인정보 보호법**
- **정보통신망 이용촉진 및 정보보호 등에 관한 법률** (정보통신망법)
- **약관의 규제에 관한 법률**

## 신고 관할

- **위치기반서비스사업 신고**: 방송통신위원회 (개인위치정보를 취급하는 경우)
- **개인정보 처리방침 관련**: 개인정보보호위원회 가이드라인 준수
- **참고 사이트**: [방통위 위치정보 통합지원 사이트](https://www.kcc.go.kr) /
  [개인정보보호위원회](https://www.pipc.go.kr)
